# Portfolio Application

This repository contains a personal portfolio website built with Angular and optional server-side rendering.

## Purpose

- Serve a portfolio UI with fast production builds.
- Provide a contact form endpoint for email submissions.
- Support deployment as a production build or a Node-based SSR app.
